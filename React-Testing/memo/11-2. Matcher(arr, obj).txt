1. 배열 비교
1-1. toEqual()
=> 배열의 모든 요소를 순서와 값으로 비교합니다.

1-2. toStrictEqual()
=> 배열의 요소뿐만 아니라, 요소의 타입이나 객체 내부까지 엄격하게 비교합니다.
=> null, undefined, 빈 배열을 다르게 인식합니다.


2. 객체 비교
2-1. toEqual()
=> 객체의 모든 키와 값을 비교. 값의 순서는 중요하지 않습니다.

2-2. toStrictEqual()
=> 객체의 키, 값, 그리고 프로토타입까지 비교합니다.

2-3. toHaveProperty()
=> 객체가 특정 키를 가지고 있는지 확인합니다.
=> 키의 값이 일치하는지 확인합니다.
=> 첫 번쨰 인자만 존재하면 키의 존재 여부를 확인합니다.
=> 두 번쨰 인자까지 존재하면 키-값의 일치를 확인합니다.


3. 특정 조건 확인
3-1. toContain()
=> 배열에 특정 값이 포함되어 있는지 확인합니다.

3-2. toContainEqual()
=> 배열 안의 객체 또는 배열 요소가 특정 값과 일치하는지 확인합니다.
