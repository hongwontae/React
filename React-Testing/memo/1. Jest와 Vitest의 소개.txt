1. Jest/Vitest 소개
=> 테스트를 찾고, 테스트를 실행하고 테스트의 실패 혹은 성공을 결정하는 라이브러리입니다.
=> Jest와 Vitest의 구문 간 차이는 없습니다.
=> 유일한 차이는 파일을 지정하는 설정과 구성입니다.


2. 명확한 차이점
=> @testing-library/react를 사용하여 컴포넌트 테스트를 작성합니다.
=> 테스트 실행 및 어설션 라이브러리로는 Jest 또는 Vitest를 선택합니다.
=> 즉, UI 테스트를 설계하는 것은 @Testing-library/react를 통해 해결하고 테스트 러너 및 모의 기능은 
Jest 혹은 Vitest로 진행합니다.
