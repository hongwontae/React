1. Jest vs Vitest
=> Vitest가 Jest보다 더 빠릅니다. (3~5)
=> Vitest는 Vite와 더 좋은 호환을 가집니다.
=> Jest는 Next.js 같은 프레임워크나 라이브러리에도 사용할 수 있습니다.
=> Jest와 Vitest는 설정만 다르고 코드는 거의 유사합니다.


2. Jest와 Vitest의 작동
2-1. test 전역 메서드
=> 첫 번쨰 인자로 테스트를 설명할 문자열, 두 번쨰 인자로 테스트가 통과할지 실패할지 실행할 함수
=> 단언을 충족하지 않으면 fail => 오류를 발생시킵니다.
