1. jotai란?
=>  React 상태 관리 라이브러리 중 하나로, atomic state를 기반으로 간단하고 
유연하게 상태를 관리할 수 있도록 설계되었습니다.


2. Jotai 주요 특징
2-1. 원자(atom) 기반 상태 관리
=> 컴포넌트는 필요한 상태(atom)만 구독하므로, 상태 변경 시 관련된 컴포넌트만 리렌더링됩니다.

2-2. 간단한 사용법
=> 사용법이 직관적이고 설정이 거의 필요 없습니다.

2-3. 비동기 상태 관리 지원
=> 비동기 데이터를 처리하기 위한 async atom 기능을 제공합니다.

2-4. React Suspense 지원
=> React의 Suspense를 통해 비동기 데이터 로딩 상태를 자연스럽게 처리할 수 있습니다.

2-5. 작고 가벼운 크기
=> Jotai는 매우 가볍고 빠른 상태 관리 라이브러리로, 프로젝트 크기를 늘리지 않습니다.
