1. 리액트 프로젝트 생성하기 : 브라우저 기반 및 로컬 개발
React Project를 만드는 두 가지 방법이 존재한다. 
하나는 로컬 시스템에 생성하는 방식이다. => 자기 컴퓨터에 React 프로젝트를 생성한다.
또 다른 하나는 브라우저 기반의 원격 리액트 프로젝트 솔루션을 이용한다. ex) 코드샌드박스를 이용한다. (react 설정이 되있는 코딩작업 브라우저를 뜻한다.)

2. 리액트 프로젝트 생성하기 : 로컬 기반
npx create-react-app 파일이름 => React가 생성된다.

npm start를 통해서 서버아래에서 프로젝트가 열린다. => 파일 프로토콜이 아니다. server+develCode
npm start의 위치는 React Project가 있는 root 위치이다.

vite라는 tool를 통해서 React 프로젝트를 생성할 수 있다.
