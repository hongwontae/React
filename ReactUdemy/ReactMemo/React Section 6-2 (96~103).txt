1. 조건 명령문 반환 추가하기
조건 명령문의 JS 로직은 따른 컴포넌트에서 관리해주는 것이 좋다.
조건 컴포넌트는 삼항문이나 조건문, 연산자들을 조합하거나 고려하여 사용해야 한다.
JSX는 JS에서 변수의 값으로 할당할 수 있다. => 컴포넌트나 HTML태그들을 변수에 할당시킬 수 있다는 것이다.


2. 연습하기 : 조건부 컨텐츠
true와 false값을 이용해 조건부 컨텐츠를 띄우는 방법이 간편하고 가독성이 높다.
&& 연산자의 특징을 이용해서 클릭 핸들러를 통해 true와 false 값을 이동시켜서 컴포넌트를 띄우거나 안띄우는 리액트 프로젝트를 생성할 수 있다. &&는 무조건 false를 찾고 false를 선택하기 때문에 click 핸들러를 통해 true로 바꾸거나 false로 바꾸면 된다.


3. 데모 앱 : 차트 추가하기
=> 이전에 했던 것처럼 동적으로 데이터를 표현하고자 한다. key와 props를 컴포넌트에 전달할 것이고 이는 map 메서드를 이용해서 이뤄질 것이다.


4. 동적 스타일 추가하기
react에서는 컴포넌트의 style 속성을 통해 css를 동적으로 표현할 수 있다. style = {{}} JSX 표현식 안의 객체안에 CSS를 구문을 적으면 된다. CSS 값에 해당하는 부분은 JS로 표현할 수 있으며 그렇지 않을 경우 따옴표로 감싸줘야 한다.

5. 마무리 및 다음 단계
