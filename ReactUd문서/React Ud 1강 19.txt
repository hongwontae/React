20. props
jsx 컴포넌트 내의 {}는 속성의 값을 부여하는 특징도 가지고 있다.
      <ExpenseItem title={expenses[0].title} amount={expenses[0].amount} date={expenses[0].date} ></ExpenseItem>
=> 만약 부모 컴포넌트에서 이렇게 속성에 값을 부여하면 자식 컴포넌트는 매개변수로 가져올 수 있다.
매개변수 명은 아무거나 해도 좋다. 이 매개변수는 속성 값을 다 가지고 있다. 리액트가 이런 작업을 자동적으로 해준다.

App.js 내의 함수 내에 배열 객체를 생성했다.
=> 이를 사용자 지정 컴포넌트에서 속성 값으로 받아들일 수 있다. => 이렇게 하는 이유는 전달하기 위해서이다.
{} => 이 특성은 키={값} 형태를 말하고 싶어서	[] => 배열 접근방식 .abcd => 객체접근방식

밑의 자식 컴포넌트에서 값을 받아들이는 것은 () 매개변수를 통해서 가져온다.
이러한 작업 리액트가 자동으로 값을 불러와주어서 가능하다.
{} => jsx 내의 js코드 실행 방법으로 값을 넣어주면 된다.
