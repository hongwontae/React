리액트-1

1. return의 문법은 jsx문법이다. JS로 바꿔준다.
2. 두개 이상의 태그는 항상 감싸줘야한다.
2-1. div로 감싸기 어려운 상황이라면 <> </> 아무런 입력도 없는 태그로 감싸주면 된다.
3. jsx 내부에 JS 변수를 보여줘야 할 때 {} 중괄호로 감싸서 보여준다.

4.Style. 인라인 스타일은 객체처럼 작성해야 하고, css에서 - 으로 구현한 css는 카멜케이스를 사용해야 한다.
5. css 파일을 따로 만들어 적용하는 방법은 css 파일을 만들고 구현체에 className=css파일이름, 이렇게 속성으로 준다.
6. jsx 내의 주석은 {/* 이런 형태로 */}작성한다.

7. props(properties) 어떠한 값을 컴포넌트에 전달해줘야 할 때 사용한다.
8. <컴포넌트 속성=""><컴포넌트/> => 속성="" 내가 아무렇게나 지정해줘도 작동한다.
9. props 또한 마찬가지다. 이름만 알기 쉽게 props로 사용한거지 내가 원하는 대로 이름을 줘도 된다. 
10. 매개변수 값으로 아무거나(가령 p)를 넣으면 p.상위 컴포넌트 속성이름를 적으면 작동하는 원리이다.
11. defaultProps => props 지정하지 않았을 때 기본적으로 사용할 값 설정
