UseEffect => 컴포넌트가 렌더링 될 떄 특정 작업을 실행할 수 있도록 하는 Hook이다

1. 첫번째 인자(effect)는 함수, 두번째 인자는 배열(deps)이 들어간다.

import react, { useEffect } from "react";
useEffect(function,deps);

effect => 수행하고자 하는 작업
deps => 배열 형태이며, 배열 안에는 검사하고자 하는 특정 값, 빈 배열

기본형태
useEffect(()=>{
console.log('마운트 될 떄마다 실행된다.')
}[]) => 컴포넌트가 화면에 가장 처음 렌더링 될 때 한 번만 실행하고 싶을 때는 deps위치에 빈 배열을 넣는다.

만약 배열을 생략한다면 리렌더링 될 때마다 실행된다.
