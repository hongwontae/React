React 1강 정리

App.js에 jsx가 public의 index.html의 id가 root인 곳에서 전달되어 실행된다.
App.js가 최상위 컴포넌트이다. 
return문이 jsx를 반환하는 구간이다.
객체를 생성하고 전달하는 방법은 props를 매개변수로 주는 것이다.
1. 객체를 생성한다. 지역변수 or 전역변수
2. 생성된 객체를 jsx의 태그의 사용자 지정 속성으로 넘겨준다.
3. 하위 컴포넌트에서 props로 받고 사용한다.

children => 상위 컴포넌트가 하위 컴포넌트를 감쌀 떄 사용하는 keyWord이다.

내가 알아야 하는 것
1. props children 활용하는 것
2. jsx가 구현되는 근본적 방식
3. 선언적 접근 방식
=> 우리가 원하는 목표 상태를 정의하고 리액트가 그 상태에 도달하도록 처리한다는 것이다.
