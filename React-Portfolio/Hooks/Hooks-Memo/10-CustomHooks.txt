1. 라액트는 use로 시작하는 함수에게 특정 규칙을 부여한다. => use로 시작하는 함수는 Hooks로 인식한다.

2. 커스텀 컴포넌트는 사용자가 만드는 Hooks이다.

3. 다른 Hooks와 이름이 겹치면 안된다.

4. use라는 이름이 붙은 함수에서 useEffect, useState를 사용해도 에러를 내뿜지 않는다.
=> 함수화하여 다른 컴포넌트에서 Hooks처럼 사용하면 된다.

5. 커스텀 훅을 한 번 사용할 떄마다 새로운 복사본이 독립적으로 만들어진다.
=> a라는 컴포넌트에서 커스텀 훅을 사용하면 해당 훅의 독립적인 상태가 만들어진 것이고
=> b라는 컴포넌트에서 동일한 커스텀 훅을 사용하면 a와 다른 독립적인 상태가 존재하는 것이다.
